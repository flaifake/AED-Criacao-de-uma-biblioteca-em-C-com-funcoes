# AED-Criação-de-uma-biblioteca-em-C-com-funções

## Objetivo do projeto
Mostrar tudo que o famoso Jukemura nos ensinou até agora, e com isso desenvolver uma biblioteca de funções de uso geral (matematica, strings e util), com finalidade de desenvolver um artefato reutilizável. Além de aprender a mexer com o GitHub.

## Como compilar e usar
1 - Baixe as bibliotecas que eu irei deixar aqui e após isso, coloque todas juntas em uma pasta.

2 - Ter um compilador para C, exemplo o "gcc".

3 - No terminal, dentro da pasta do projeto, execute
```
gcc main.c matematica.c strings.c util.c -o output\main.exe
.\output\main.exe 
```
4 - O arquivo main.c mostra um exemplo de uso de todas as funções da biblioteca.
